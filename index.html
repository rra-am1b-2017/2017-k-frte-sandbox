Output

We gaan een pop-up maken met javascript. We drukken op een knop en er verschijnt een pop-up
Je kunt tekst naar het browserscherm schrijven met document.write
We kunnen ook naar de console schrijven van de browser. Dit doen we met console.log("");
Je kunt ook tekst inlezen in een input tag door het drukken op een knop en deze weergeven in een paragraaf tag.
Opdracht
  1. maak een input-tag met id
  2. maak een button
  3. maak een paragraaf tag met een id
  4. door het drukken op de button wordt wat je ingevuld hebt in de paragraaf tag weergegeven.

Arrays

Je kunt met javascript ook indexed arrays maken. een indexed array ziet er zo uit:

var fruit = ["sinaasappel", "appel", "meloen"]

Door op de onderstaande knop te drukken wordt het array hiernaast getoond en ook in de console van de browser.
